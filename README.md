# De Pátria para Pátria

## Descrição do Projeto

Uma jornada épica pelas cidades natais dos funcionários da TripleTen, do Kentucky ao Burundi, passando pelo País de Gales e Ucrânia. Este projeto apresenta uma galeria de arte digital que celebra a diversidade cultural e as histórias únicas de pessoas de diferentes cantos do mundo.

O site foi desenvolvido como uma página responsiva que se adapta perfeitamente a diferentes dispositivos, oferecendo uma experiência visual imersiva sobre as origens e culturas dos colaboradores da TripleTen.

## Funcionalidades

- **Design Responsivo**: Adaptação perfeita para desktop (1280px), tablet (768px) e mobile (320px)
- **Layout Flexível**: Utilização de Flexbox e CSS Grid para organização dos elementos
- **Tipografia Otimizada**: Implementação da fonte Inter para melhor legibilidade
- **Experiência Visual**: Interface limpa e moderna focada no conteúdo

## Tecnologias e Técnicas Utilizadas

- **HTML5**: Estrutura semântica com tags apropriadas
- **CSS3**: Estilização avançada com:
  - Flexbox para layouts flexíveis
  - CSS Grid para organização de conteúdo
  - Media Queries para responsividade
  - Metodologia BEM para organização do CSS
- **Design Responsivo**: Breakpoints em 544px e 1024px
- **Figma**: Utilizado como referência para implementação fiel do design
- **Git/GitHub**: Controle de versão e hospedagem
